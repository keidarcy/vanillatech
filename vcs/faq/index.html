What does Vanilla Cross Sell APP do?

We help you create an integrated cross sell solution for your store with just a few clicks.

We added many different Cross sell and upsell to many Shopify pages.Relevant recommendations will be shown on each page for best results:

  • Best Selling Products: Home page, Cart page
  • Newest Products: Home page, Cart page
  • Recommended Products: Product pages
  • Recently Viewed Products: All Shopify pages


Which theme will be added?

Only the current theme will be added.
If you publish a new theme, please save setting in Vanilla Cross Sell APP again.



How to uninstall the Vanilla Cross Sell APP?

There are only two steps you need to do.

  1. In order to delete all our code please go into your Store admin - Themes - delete snippets/vcs.liquid.
  2. layout/vcs_back_theme_{timestamp}.liquid is the layout/theme.liquid before Vanilla Cross Sell APP edited.
